# National Parks Explorer

## Description

National Parks Explorer is an iOS application that displays a list of ten U.S. National Parks using a TableView. Selecting a park navigates to a detail screen that displays additional information about the selected park.

## Features

- UITableView
- Multiple ViewControllers
- Segue Navigation
- Data Passing Between Controllers
- MVC Architecture
- Supports iPhone and iPad
- Landscape Orientation

## Technologies

- Swift
- UIKit
- UITableView
- Storyboard
- MVC Design Pattern
